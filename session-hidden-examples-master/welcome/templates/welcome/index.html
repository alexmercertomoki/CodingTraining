This example include three applications:
  • hidden -- an example showing the use of hidden fields
  • shared -- a "shared" to do list example. All users will see the same list.
  • private -- a "private" to do list example. Each browser instance will have it's own to do list.

Note: The private to do list example uses sessions. The default implementation of sessions in Django stores the sessions in the sqlite database. To make this work, you will need to execute the following command to set up the database: 


        python manage.py migrate
before executing: 

        python manage.py runserver
